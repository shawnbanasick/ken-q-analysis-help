How to add data

User Input

MS Excel file

Ken-Q Analysis can import Excel files in three formats

  • Type 1 - Q-sort data arranged by Q-sort value
  • Type 2 - Q-sort data arranged by statement number
  • Type 3 - Ken-Q Analysis output file created during a previous session

For projects with unforced (non-symmetric) Q-sorts, use the Excel Type 2 format or a PQMethod DAT file.

Type 1 - This format is the best choice when importing hand-written Q-sort information with forced (symmetric) Q-sorts.
